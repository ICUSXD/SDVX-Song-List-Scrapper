# SDVX-Song-List-Scrapper

Just a simple Scrapper from https://p.eagate.573.jp/game/sdvx/vi/music/index.html<br/>
Saved as json

Structure:<br/>

{<br/>
  "level": {<br/>
    "index": {<br/>
      "title": song title,<br/>
      "artist": artist,<br/>
      "jacket": jacket image link<br/>
    }<br/>
}
